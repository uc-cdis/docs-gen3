# A new blog series: How does Gen3...

At CTDS, we field many questions from beginner (and even experienced!) Gen3 users asking how Gen3 manages different data management and user access tasks. We have decided to start a new blog series to tackle some of these topics. You should be able to find these posts at [https://docs.gen3.org/blog/category/how-does-gen3/](https://docs.gen3.org/blog/category/how-does-gen3/).
