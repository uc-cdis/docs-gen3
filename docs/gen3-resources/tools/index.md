# Gen3 Tools

The Gen3 team has created a number of stand alone command line tools to help users or researchers interact with data (structured and unstructured) in a data commons.  The two primary tools that users may use include the Gen3 Client and the Gen3 Python SDK.  A quick comparison of the two can be found in the following table.



| Feature | Data Client | Python SDK |
|---|---|---|
| Command Line Tool | Yes | Yes |
| Upload and Index Data Files | Yes | Yes |
| Download Data Files | Yes | Yes |
| Query Structured Data | No | Yes |
| Submit Structured Data | No | Yes |
| Query Semistructured Data | No | Yes |
| Submit Semistructured Data | No | Yes|
| Mapping Files to a Data Node | No | Yes|
