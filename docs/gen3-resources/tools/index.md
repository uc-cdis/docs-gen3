# Gen3 Tools

The Gen3 team has created a number of stand alone command line tools to help users or researchers interact with data (structured and unstructured) in a data commons.  The two primary tools that users may use include the Gen3 Python SDK and the Gen3 Data Client.  The Gen3 Python SDK is a collection of Python functions that make it easier to interact with Gen3's APIs programmatically including for file download.  The Gen3 Data Client is a tool for uploading or downloading files from a data commons.
